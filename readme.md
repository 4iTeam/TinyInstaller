# Tiny installer
## Yêu cầu
Hệ điều hành Ubuntu 18,20 hoặc Debian 9,10
## Lệnh cài đặt

Copy lệnh bên dưới nhớ chạy bằng root nhé

```console
wget https://github.com/4iTeam/TinyInstaller/raw/main/ti.sh
bash ti.sh "link tới file gz"
```
